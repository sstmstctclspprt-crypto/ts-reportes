Tactical Support

Digitalización de inspecciones C-TPAT
Sin papel PDF al instante Trazabilidad Respaldo en nube
1 ¿En qué consiste?

Tactical Support es una aplicación web (PWA) para digitalizar inspecciones C-TPAT en patio: checklists, evidencias fotográficas, firmas digitales y reportes PDF con folio consecutivo, sin formatos físicos.

Antes — Proceso en papel
Formatos físicos extraviados
Información incompleta o ilegible
Reportes tardíos o perdidos
Sin trazabilidad ni respaldo
Ahora — Proceso digital
Todo integrado en una sola app
PDF generado al guardar
Respaldo automático en nube
Trazabilidad con folio TS-n
Vercel — App PWA Supabase — Auth · BD · Storage Google — OAuth · Drive Microsoft — OneDrive respaldo
2 ¿Cómo funciona?

El operador inicia sesión, captura la inspección completa en el dispositivo y, al guardar, el sistema genera el PDF y lo respalda en la nube.

5 Procesos y funciones — cómo se mueve la app

Rutas principales

/ Inicio — lista registros, config logo y OneDrive
/activacion Canjear código de acceso
/admin Panel admin — códigos y usuarios
/registro/new Nueva inspección C-TPAT
/registro/:id Editar registro existente

Guardado del registro

RegistroForm valida datos (Zod) y sube fotos/firmas a Storage
RPC next_folio_ctpat asigna folio TS-0001…
INSERT en registros_ctpat (sync_status: pending)
syncStore encola generación PDF y procesa cola
Sin internet: cola offline (IndexedDB) al reconectar

Generación PDF (Edge Function)

POST generate-ctpat-pdf con JWT del usuario
Lee registro, logo y evidencias desde BD/Storage
Arma PDF (4 páginas) y sube a Google Drive
Backup en Storage + webhook Power Automate → OneDrive
UPDATE sync_status=synced · datos sensibles expiran a 7 días

Operación continua

HomeView muestra estado de sync por registro
Reintento manual si falla subida a Drive
Cron purge-expired-ctpat-data elimina datos vencidos
GitHub Actions: migraciones BD + deploy functions
Vercel: build y CDN de la PWA en cada push
